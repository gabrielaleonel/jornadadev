# Exercício 1 — Função ou Procedimento?

| Item | Tipo | Justificativa |
|------|------|---------------|
| Str() | Função | Retorna uma string convertida a partir de um número. |
| QOut() | Procedimento | Apenas exibe texto na tela (efeito colateral), não retorna valor. |
| Date() | Função | Retorna a data atual do sistema como valor. |
| Len() | Função | Retorna o tamanho (número inteiro) de uma string ou array. |
| Upper() | Função | Retorna a string convertida para maiúsculas. |
| ClearScreen() | Procedimento | Apenas limpa a tela (ação visual), não retorna valor. |

## Regra geral
- **Função** = retorna um valor que pode ser usado em expressões ou atribuições.
- **Procedimento** = executa uma ação sem retornar valor (RETURN NIL ou sem RETURN).
