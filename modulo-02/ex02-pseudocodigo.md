# Exercício 2 — Pseudocódigo

## a. Calcular a área de um retângulo (base × altura)

```
Início
    Leia base
    Leia altura
    area ← base * altura
    Escreva area
Fim
```

## b. Verificar se um número é par ou ímpar

```
Início
    Leia numero
    Se (numero MOD 2 = 0) então
        Escreva "par"
    Senão
        Escreva "ímpar"
    FimSe
Fim
```

## c. Encontrar o maior entre três números

```
Início
    Leia a
    Leia b
    Leia c

    Se (a >= b) e (a >= c) então
        Escreva a, " é o maior"
    Senão, se (b >= a) e (b >= c) então
        Escreva b, " é o maior"
    Senão
        Escreva c, " é o maior"
    FimSe
Fim
```
